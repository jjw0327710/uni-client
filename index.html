기능

필요한 건 다 있습니다

마인크래프트를 즐기는 데 필요한 기능들을 한 곳에서.

Microsoft 계정 로그인

브라우저 팝업으로 빠르게 로그인. 정품 인증 완료된 계정이면 바로 시작.

Fabric 자동 설치

프로파일마다 Fabric을 자동으로 설치. 버전 선택부터 실행까지 원클릭.

성능 최적화

Sodium + Lithium 모드를 원클릭 설치. 저사양 PC에서도 쾌적하게.

좌표·방향 표시

게임 내 좌표와 바라보는 방향을 화면에 표시. 모드 자동 설치.

Simple Voice Chat

인게임 음성 채팅 모드 자동 설치. 친구들과 바로 대화하세요.

스크린샷

F2로 찍은 스크린샷을 런처에서 바로 확인. 폴더 열기도 한 번에.
